# Hospital Performance & Readmission Analytics System

## 📌 Project Overview
This project focuses on analyzing hospital operational efficiency, patient treatment patterns, and insurance claims, with a primary objective of understanding and reducing patient readmission rates. By integration of SQL-based data management and Power BI visualizations, this system provides actionable insights for healthcare administrators to improve patient care and optimize resource allocation.

## 🛠️ Tech Stack & Tools Used
*   **Database & Querying:** SQL (Structured Query Analysis for data extraction and joining)
*   **Data Visualization:** Power BI (Interactive Dashboards & Analytics)
*   **Documentation:** Microsoft PowerPoint (Project Presentation)
*   **Data Source:** Cleaned CSV datasets containing patient master records, hospital operations, insurance claims, and treatment summaries.

## 📂 Repository Contents
*   `hospital Capstone project (1).sql`: SQL queries used for data manipulation and analysis.
*   `power bi file final projrct (1).pbix`: Power BI Dashboard file containing interactive reports.
